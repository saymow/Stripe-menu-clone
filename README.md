## :clipboard: How to use?

```
# Clone this repositoy
$ git clone git@github.com:saymow/Stripe-menu-clone.git

# Change into its directory
$ cd Stripe-menu-clone

# Install dependencies
$ npm install

# Run application.
$ npm start
```
